## 1. 다나와 Gap 분석

### 다나와 핵심 기능 vs 현재 상태

| 기능 | 다나와 | 현재 프로젝트 | Gap |
|------|--------|--------------|-----|
| 가격 추이 차트 | ✅ 제품별 라인 차트 | API 있음, UI 없음 | **크리티컬** |
| 역대 최저가 배지 | ✅ "지금이 최저가" | 없음 | **크리티컬** |
| 인기 랭킹 TOP 100 | ✅ 카테고리별 TOP 100 | 없음 | **높음** |
| 멀티채널 가격 비교 | ✅ 쇼핑몰별 가격 비교 | ✅ /compare, /compete | 완료 |
| 검색 자동완성 | ✅ 실시간 드롭다운 | 관련검색어만 있음 | **높음** |
| SEO (구글 유기 트래픽) | ✅ 강력 | 메타태그 거의 없음 | **높음** |
| 가격 알림 구독 | ✅ 목표가 알림 | Discord만 있음 | **중간** |
| 카테고리 탐색 | ✅ 계층적 메뉴 | 필터 있음, 탐색 약함 | **중간** |
| 리뷰/커뮤니티 | ✅ 사용자 리뷰 | 없음 | 미래 과제 |

### 패션 다나와만의 차별화 강점 (다나와에 없음)

| 기능 | 설명 | 현재 상태 |
|------|------|----------|
| Intel Hub | 산업 인텔리전스 (뉴스/드롭/스파이크) | ✅ 구현됨 |
| 브랜드 스토리 | 디렉터, 협업, 역사 | ✅ 구현됨 |
| 글로벌 채널 | JP, US, EU 포함 | ✅ 구현됨 |
| 드롭 캘린더 | 출시 예고 캘린더 | API 있음, UI 약함 |
| 세일 히트맵 | 브랜드×카테고리 세일 강도 | 없음 |

## 3. 비전

```
현재: 패션 가격 수집 + 세일 목록 + 인텔 허브
목표: 한국 패션 가격비교 표준 플랫폼

포지셔닝:
  다나와 = 가전의 최저가 성경
  패션 다나와 = 패션의 최저가 + 트렌드 인텔리전스

핵심 가치:
  1. "지금 이 제품, 여기서 사는 게 맞아?" (가격 신뢰)
  2. "다음 드롭은 언제야?" (정보 선점)
  3. "이 브랜드 지금 얼마나 세일 해?" (인텔리전스)
```
